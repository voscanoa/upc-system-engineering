# Apuntes de Ingeniería de Sistemas - UPC

Repositorio personal con todos mis apuntes, resúmenes y materiales de estudio de la carrera de Ingeniería de Sistemas en la Universidad Peruana de Ciencias Aplicadas (UPC).

Este repositorio tiene como objetivo centralizar y organizar todo el conocimiento adquirido durante mi formación académica, facilitando el repaso y sirviendo como material de consulta rápida.
