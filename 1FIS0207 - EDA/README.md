# Estructura de Datos y Algoritmos – 1FIS0207

Este repositorio reúne todo lo que desarrollé durante el curso **Estructura de Datos y Algoritmos (1FIS0207)**. Aquí organizo mis prácticas, apuntes, proyectos y ejercicios relacionados con algoritmos, estructuras de datos y resolución de problemas.


## ¿Qué encontrarás aquí?

He organizado el repositorio por unidades, siguiendo el contenido oficial del curso:

### **Unidad 1: Fundamentos de Algoritmos y Programación**

Incluyo:

* Algoritmos básicos
* Estructuras secuenciales, condicionales y repetitivas
* Manejo de cadenas
* Ejercicios introductorios en programación


### **Unidad 2: Estructuras de Datos Básicas**

Trabajo con:

* Arreglos
* Listas
* Pilas
* Colas
* Operaciones fundamentales y representación en memoria


### **Unidad 3: Recursividad**

Aquí explico e implemento:

* Tipos de recursividad
* Recursividad de cola
* Diferencias entre recursivo vs iterativo
* Ejercicios aplicados

### **Unidad 4: Algoritmos de Ordenamiento y Búsqueda**

Implemento y comparo:

* Burbuja
* Inserción
* Selección
* Quicksort
* Búsqueda lineal y binaria

## Herramientas que utilicé
